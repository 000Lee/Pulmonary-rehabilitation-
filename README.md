## 1. 호흡기재활이란 (비의료인용)

- **✅ 확정** — 국제 표준 정의(미국흉부학회 ATS·유럽호흡기학회 ERS, 2013): 폐질환 환자를 철저히 평가한 뒤 그 환자에 맞춘 운동훈련·교육·행동변화를 포함하는 종합적 중재로, 신체적·심리적 상태를 개선하고 건강한 생활습관을 오래 유지하도록 하는 것. 〔근거: [ATS/ERS Official Statement, *AJRCCM* 2013](https://www.atsjournals.org/doi/full/10.1164/rccm.201309-1634ST)〕
- **쉬운 설명**: 폐나 호흡근육이 약해진 환자가 "숨이 덜 차게, 더 오래 움직일 수 있게" 만드는 재활. 약을 주는 게 아니라 근육 훈련·객담 배출·운동을 조합해 호흡 능력 자체를 끌어올림.
- **✅ 확정** — 근거 수준은 COPD(만성폐쇄성폐질환)에서 가장 강함(운동능력·호흡곤란·삶의 질·입원 감소). 단, 이 근거는 주로 **안정기 만성폐질환** 대상임을 발표 시 명확히 할 것. 〔근거: 위 ATS/ERS Statement 동일〕

## 2. 실제로 어떤 "행위"를 하는가

한국 임상 시행 단계 〔근거: [대한심장호흡물리치료학회 공지](https://www.kacrpt.org/community/notice/view?aid=3481), 대한결핵 및 호흡기학회 『호흡재활 지침서 2015』 기반〕

- **✅ 확정** — **초기 평가**: 폐기능검사(Spirometry), 기침유량 측정(Peak Flow Meter), 호흡근력 검사(MIP/MEP), 산소포화도(Pulse-Oximeter).
- **✅ 확정** — **치료 기법**: 객담 배출(도수기법 또는 Cough Assist 기기), 호흡근 훈련(흡기근/호기근 강화 기기), 유산소 훈련(고정식 자전거·트레드밀), 호흡 패턴 교정 및 자가관리 교육.
- **비의료인용 풀이**:
  - *객담 배출* = 가래를 못 뱉는 환자에게 손으로 흉곽을 두드리거나 기계로 기침을 유도해 가래를 빼냄
  - *호흡근 훈련* = 들숨에 저항이 걸리는 기구를 물게 해 아령 운동처럼 호흡근육을 단련
  - *유산소 훈련* = 자전거·걷기로 전신 지구력을 올려 숨이 덜 차게 함
- **⚠️ 미확정** — 위 초기평가 4가지를 **전부 반드시** 해야 산정되는지(강제 목록 여부)는 심평원 원문 미확인. 학회 자료상 "등"으로 표기된 예시임.

## 4. 인력·산정 요건 (이 사안의 핵심)

- **✅ 확정** — 호흡재활치료는 **사-129, 전문재활치료료** 항목임. 〔근거: [심평원 심사기준조회 청구내역례](https://m.hira.or.kr/mobile/rf/iac/eva/data.do?edate=&matrSeqNo=1&openDT=20170831&rnum=&sdate=&seqNO=6)〕
- **✅ 확정** — **전문재활치료료는 재활의학과 전문의가 상근**하고, 그 전문의(또는 전공의) 처방에 따라 상근 물리치료사(또는 해당분야 전문치료사)가 실시하고 결과를 진료기록부에 기록한 경우 산정. **재활의학과 전문의 휴가 중에는 산정 불가**. 〔근거: [심평원 고객의 소리 급여기준 FAQ (2024-09-13)](https://www.hira.or.kr/bbsDummy.do?pgmid=HIRAA010006011100&brdScnBltNo=4&brdBltNo=47554)〕
- **✅ 확정** — 호흡재활치료(사-129)는 **상근 재활의학과 전문의 처방**이 있어야 산정 가능한 전문재활치료료 항목. 〔근거: [한국노동연구원 『산재보험 의료재활수가체계 개선방안 연구』](https://dl.kli.re.kr/library/10320/contents/6021544)〕
- **✅ 확정** — **귀원은 재활의학과 전문의 상근 → 이 관문 충족.** (흉부외과 전문의만으로는 사-129 산정 불가였음 → 재활의학과 상근으로 해결)
- **✅ 확정** — **상근 요건 위반 시 급여환수 리스크**: 재활의학과 전문의가 주 1~3일만 근무한 병원이 전문재활치료료를 청구했다가 "상근 아님"으로 **부당이득 급여환수 처분이 정당**하다는 판결. 〔근거: [서울행정법원 판결, 메디포뉴스 보도](https://www.medifonews.com/news/article.html?no=72435)〕
- **✅ 확정** — 물리치료사의 **교육 수료증은 산정요건이 아님**. 실시자는 "상근 물리치료사 또는 해당분야 전문치료사"면 됨. (학회의 이수증 권고는 법정요건이 아닌 실무 권장) 〔근거: 위 [심평원 FAQ (2024-09-13)](https://www.hira.or.kr/bbsDummy.do?pgmid=HIRAA010006011100&brdScnBltNo=4&brdBltNo=47554)〕

## 5. 청구 실무 디테일

- **✅ 확정** — **MM290은 [1일당] 정액 코드**(상대가치 약 83.67점). 그날 매뉴얼을 하든 기구를 하든 둘 다 하든 **하루 1회 같은 금액**. 기법 수가 늘어도 더 붙지 않음. 〔근거: [KIHASA 『보건사회연구』 42(1), 2022, p.110 (심평원 자료 인용)](https://www.kihasa.re.kr/hswr/assets/pdf/1303/journal-42-1-106.pdf)〕
- **✅ 확정** — 단, 코드 유의사항이 **"기구 병행 필수"**이므로 **매뉴얼만 단독 시행은 산정요건 미충족 소지**. 〔근거: [대한심장호흡물리치료학회](https://www.kacrpt.org/community/notice/view?aid=3481)〕
- **✅ 확정** — **흉부 매뉴얼 기법(타진·진동·체위배출)은 MM290 안에 포함**된 행위(객담관리). 별도 코드로 따로 청구하는 것이 아님. 〔근거: 위 [대한심장호흡물리치료학회](https://www.kacrpt.org/community/notice/view?aid=3481)〕
- **✅ 확정** — **도수치료(급여 코드)는 근골격계 질환 전용**(30분 이상, 주2회·연15~24회 제한, 2026.7.1. 관리급여 전환). **호흡·심폐 목적으로는 도수치료 코드 청구 불가**. 〔근거: [보건복지부 도수치료 관리급여 보도자료](https://www.mohw.go.kr/board.es?mid=a10503010100&bid=0027&act=view&list_no=1491109)〕
- **✅ 확정** — 기록 요건은 **"결과를 진료기록부에 기록"**만 요구. PN 노트든 별도 치료기록지든 서식 무관하나, 삭감 방어 관점에선 시행내용이 구체적인 치료기록지가 안전. 〔근거: 위 [심평원 FAQ (2024-09-13)](https://www.hira.or.kr/bbsDummy.do?pgmid=HIRAA010006011100&brdScnBltNo=4&brdBltNo=47554)〕

### 신경계 치료와의 관계 (별도 산정 여부)

- **✅ 확정** — 호흡재활은 NDT·mat·FES와 **다른 계열의 별도 코드**. (NDT=사-122 중추신경계발달재활치료 / mat=사-130 재활기능치료 / 호흡재활=사-129) "1종만 인정" 제한은 **같은 계열 안에서만**(예: 매트+보행 중 1종) 적용됨. 〔근거: [대한재활의학과의사회 전문재활치료 인정기준](https://www.rm.or.kr/?c=3%2F22&uid=3635)〕
- **✅ 확정** — **실제 청구례**: 한 환자에게 사-122 중추신경계발달재활치료 + 사-129 호흡재활치료 + 서-141 연하장애재활치료가 **함께 청구**된 심사사례 존재 → 신경계 치료 환자에게 심폐 적응증이 있으면 호흡재활이 **별도로 추가되는** 구조가 실제 작동. 〔근거: [심평원 심사기준조회 청구내역례](https://m.hira.or.kr/mobile/rf/iac/eva/data.do?edate=&matrSeqNo=1&openDT=20170831&rnum=&sdate=&seqNO=6)〕
- **⚠️ 미확정** — 위 사례가 "같은 날" 동시 청구인지, 날짜를 나눈 것인지는 요약만으로 확정 불가. "환자 단위 추가"는 확실하나, **같은 날 동시산정 시 심사조정 여부**는 원문 확인 필요.

## 6. 환자 상태별 실제 시행 (침상 수동 → 기구 운동)

> **핵심 구분**: "중환자실이냐"가 아니라 **환자가 지시를 따르고 능동적으로 움직일 수 있느냐(협조도)**로 갈림. 같은 환자도 회복되면 침상 → 기구로 단계 이행함.
>
> **🔑 청구 관점(가장 중요)**: 아래 어느 단계든 **MM290 산정에는 "기구 병행"이 필수**임. 흉부 타진·진동 같은 매뉴얼(손기술)만 단독으로 하면 산정요건 미충족 소지. **매뉴얼을 하더라도 Cough Assist·흉벽진동기·흡기근 훈련기 등 기구가 함께 들어가야 청구가 안전함.** 〔근거: [대한심장호흡물리치료학회 "기구를 이용한 치료가 병행되어야 함"](https://www.kacrpt.org/community/notice/view?aid=3481)〕

### (A) 능동 참여 가능 (협조 됨) — 심폐 운동기구 중심 (MM440 영역)

- **✅ 확정** — **유산소 기구**: 고정식 자전거(에르고미터), 트레드밀, 상지 에르고미터(팔 자전거, 하지 약한 환자용). 〔근거: [대한심장호흡물리치료학회(유산소: 고정식 자전거·트레드밀)](https://www.kacrpt.org/community/notice/view?aid=3481)〕
- **✅ 확정** — **호흡근 강화 기구**: 흡기근 훈련기(IMT, threshold/resistive), 호기 양압 기구(PEP/flutter). 〔근거: 위 [대한심장호흡물리치료학회(호흡근 훈련기)](https://www.kacrpt.org/community/notice/view?aid=3481)〕
- **근력**: 탄력밴드·소도구 웨이트, 좌위·기립 자세 저항운동
- **🔑 청구**: 이 단계는 기구가 주(主)이므로 기구 병행 요건 자연 충족.

### (B) 능동 참여 불가 (인지저하·진정·기관삽관으로 협조 안 됨) — 침상(베드) 수동 중심 (MM290 영역)

- **✅ 확정** — **객담 관리**: 흉부 타진·진동, 체위배출(자세 잡아 가래 빼기), **Cough Assist(기계로 기침 유도)**, 필요시 **고빈도 흉벽진동(MM430)**. → 여기서 Cough Assist·흉벽진동기가 "병행 기구" 역할. 〔근거: [대한심장호흡물리치료학회(객담배출: Manual technique 또는 Cough Assist)](https://www.kacrpt.org/community/notice/view?aid=3481)〕
- **수동 관절운동(PROM)**: 사지 구축 예방 위해 치료사가 관절을 움직여줌
- **체위 변경·자세잡기**: 폐 환기 좋은 자세, 무기폐·욕창 예방
- **기립 적응**: 틸트테이블·스탠딩프레임으로 단계적 기립 자극(혈압·산소포화도 모니터링). ※ 스탠딩프레임·틸트는 "심폐 운동기구"라기보다 **능동 이행 전 단계의 기립 적응 도구**임.
- **수동적 순환 자극**: 침상 사이클(모터로 다리를 돌려주는 기구) 등
- **🔑 청구(반복 강조)**: 이 단계는 손기술 비중이 크므로 **의식적으로 기구(Cough Assist·흉벽진동기·침상 사이클 등)를 병행**해야 MM290 산정 안전. 매뉴얼만 기록되면 삭감 위험.

### 단계 이행 개념

- **✅ 확정** — 협조도에 따라 **침상 수동 → 침상 능동 → 좌위 → 기립 → 기구 운동** 순으로 상향. 회복 시 (B)에서 (A)로 이행.
- **⚠️ 미확정** — (A)/(B) 각 행위가 정확히 어느 코드(MM290/MM440/MM430)로 산정되는지의 세부 매핑과, 침상 수동기법 단독의 산정 가부는 심평원 원문 확인 필요.

## 7. 적응증·대상 환자·금기

〔근거: [대한심장호흡재활의학회지, *Ann Cardiopulm Rehabil* 1(1):23](https://www.eacpr.org/archive/view_article?pid=acpr-1-1-23)〕

- **✅ 확정** — 적절한 약물치료에도 **증상이 지속·악화되거나, 잦은 병원 방문이 필요하거나, 활동 제한이 있는** 환자는 호흡재활을 반드시 고려. 대표 대상은 COPD·천식이며, 비-COPD 질환에서도 운동내성·삶의 질 개선 근거가 축적 중.
- **대상 환자군(요양병원 현실 기준)**:
  - **✅ 확정** — 만성폐질환: COPD, 천식, 기관지확장증, 간질성폐질환
  - **✅ 확정** — 신경근육질환으로 인한 호흡부전(ALS, 근이영양증 등)
  - **✅ 확정** — **신경계 환자의 호흡 합병증**: 뇌졸중·척수손상 환자에서 흔한 호흡근 약화·객담 배출 곤란·흡인성 폐렴 위험 → **기존 신경계 환자군에서 적응 대상 다수 발생(발표 핵심 논리)**
  - 흉부·상복부 수술 전후, 인공호흡기 이탈(weaning) 과정
- **✅ 확정** — **금기(발표 시 반드시 언급)**: 불안정 협심증, 급성심근경색, 심한 대동맥판 협착증, 치료하지 않은 중증 폐동맥고혈압은 운동치료 금기. 안정화된 심장질환·폐동맥고혈압은 의료진 협의 하 개별화 처방 가능. 최종 대상자 선정은 임상전문의와 호흡재활팀 판단. 〔근거: 위 [대한심장호흡재활의학회지](https://www.eacpr.org/archive/view_article?pid=acpr-1-1-23)〕
- **✅ 확정** — **상병만 붙인다고 자동 인정 아님**: '외상성 경막하출혈 + 흡인성 폐렴' 상병의 호흡재활치료가 "인정기준 미달"로 조정(삭감)된 실제 사례 → 호흡재활의 의학적 필요성 근거가 기록에 있어야 함. 〔근거: [심사 2017-바-8 진료비 심사결정 사례집](http://file.ltoss.co.kr/updata/newout/upload/185/170922140559000001989/근로복지공단-2017진료비심사결정사례집(2015년-2016년)170630.pdf)〕

## 8. 심평원 원문 확인 필요 — 미확정 체크리스트

발표 전 아래 항목은 **심평원 요양기관업무포털 > 급여기준 조회**에서 조문 대조 필요:

- **⚠️ 미확정** — **MM440(폐질환 운동재활치료) 정확한 산정기준·장비요건** (웹 공개자료로 조문 미확보)
- **⚠️ 미확정** — **MM290 ↔ MM440 같은 날 동시산정 가능 여부** (실무상 가장 중요)
- **⚠️ 미확정** — **MM290의 필수 평가·최소 기구 강제 목록**
- **⚠️ 미확정** — **급여 대상 상병코드의 구체적 범위**
- **⚠️ 미확정** — **사-44 만성호흡부전 재활치료** 코드 세부 요건
- **확인 경로**: 상근 재활의학과 전문의 또는 원무·심사팀이 사-129 / MM440 / 사-44 조문을 조회 → 동시산정·상병범위 일괄 정리 가능

## 9. 발표용 요약 (확정 사실만)

- 호흡재활치료(MM290)는 전문재활치료료 → **재활의학과 전문의 상근이 산정 필수요건, 귀원 충족**
- 상근 요건 위반 시 **급여환수 판례** 존재 (주의)
- 물리치료사 교육 수료증은 산정요건 아님
- MM290은 [1일당] 정액 → 기법 수 무관 같은 금액, **단 기구 병행 필수**(매뉴얼 단독 불가 소지)
- **환자 협조도로 갈림**: 협조 됨 → 심폐 운동기구(자전거·트레드밀·IMT), 협조 안 됨 → 침상 수동(객담관리·PROM·기립적응). **어느 단계든 MM290 청구엔 기구 병행 필수**
- 흉부 매뉴얼은 MM290에 포함 / 도수치료 코드는 근골격계 전용(호흡목적 청구 불가)
- 호흡재활은 NDT·mat·FES와 **별개 코드로, 심폐 적응증 있으면 환자에게 추가 산정**(실제 청구례 확인)
- 대상: 만성폐질환 + 신경근육질환 호흡부전 + **신경계 환자의 호흡 합병증**(기존 환자군 활용)
- 금기: 불안정 협심증·급성심근경색·중증 대동맥판 협착·미치료 중증 폐동맥고혈압
